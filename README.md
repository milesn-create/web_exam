#  Языковая школа - Экзаменационный проект

Веб-сайт для записи на языковые курсы и к репетиторам.  
Разработан в рамках экзамена по дисциплине "Основы веб-технологий".

##  Основные функции
- Главная страница с курсами и репетиторами
- Личный кабинет с управлением заявками
- Расчет стоимости с учетом опций
- Интеграция с REST API

##  Технологии
- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5.3
- REST API, Fetch API

## Проверка кода
- HTML проверен через HTMLHint
- JavaScript проверен через ESLint
